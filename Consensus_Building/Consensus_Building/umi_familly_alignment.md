# UMI family alignment 

**Tool:** MAFFT version 7.453

Parameters used:

### **--localpair**

Uses a sensitive alignment strategy, improving alignment accuracy.

### **--maxiterate 500**

Performs iterative refinement for up to 500 cycles to optimize the multiple sequence alignment and improve overall alignment quality.

Below are a few html files which can easily open interactively in a new browser tab.

The aligned fasta sequence files can be directly downloaded by clicking the fasta links.
